# 🚀 快速启动指南

## 推荐方式：双击 `start.bat` 文件

### 🎯 最简单的启动方法

1. **打开文件夹** `e:\OpenCode\SG`
2. **双击** `start.bat` 文件
3. **等待** 浏览器自动打开应用（首次可能需要安装依赖，需要1-2分钟）

就这么简单！✨

---

## 📋 其他启动方式

### 方式1：使用 PowerShell 脚本

双击 `start.ps1` 文件（功能同 `.bat`，但显示更漂亮）

**注意**：如果提示 PowerShell 不允许执行脚本，请先运行：
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

---

### 方式2：创建桌面快捷方式

#### 在 Windows 上创建快捷方式：

1. **右键点击** `start.bat` 文件
2. **选择** "发送到" → "桌面（创建快捷方式）"
3. **右键点击** 桌面上的快捷方式
4. **选择** "属性"
5. **修改快捷方式选项**（可选）：
   - **名称**：改为 "AI 提示词生成器" 或其他
   - **快捷键**：设置快捷键（如 Ctrl+Shift+P）
   - **运行方式**：改为"最小化"（可选）

**设置图标（可选）**：
- 右键快捷方式 → 属性 → 高级 → 改为"以管理员身份运行"
- 点击"更改图标"设置自定义图标

---

### 方式3：命令行启动

打开 PowerShell 或 cmd，运行：
```bash
cd e:\OpenCode\SG
npm run dev
```

## ⚠️ 常见问题

### ❌ "找不到 Node.js"

**解决方案**：
1. 下载 Node.js v18+ 从 https://nodejs.org
2. 安装时勾选 "Add to PATH"
3. 重启电脑
4. 再次运行 `start.bat`

---

### ❌ 脚本无法执行（PowerShell）

**解决方案**：
1. 以管理员身份打开 PowerShell
2. 运行：`Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`
3. 输入 `Y` 确认

---

### ❌ 第一次运行很慢

这是**正常的**！第一次需要：
- 下载 npm 包（可能需要1-2分钟）
- 编译 TypeScript 和 Tailwind CSS

后续启动会更快。

---

## 💡 停止应用

在运行脚本的终端窗口中：
- 按 **Ctrl+C** 停止开发服务器
- 关闭终端窗口

---

## 🌐 访问应用

应用运行后会自动打开浏览器，访问：
```
http://localhost:5173
```

如果没有自动打开，手动打开浏览器访问上述地址。

---

## 📦 重新安装依赖

如果遇到依赖问题，可以：

1. **删除** `node_modules` 文件夹
2. **删除** `package-lock.json` 文件
3. **运行** `start.bat`（会自动重新安装）

---

## 🔄 更新代码后

如果修改了代码，应用会**自动热更新**，无需重启。
